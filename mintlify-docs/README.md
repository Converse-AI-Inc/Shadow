# Shadow Mintlify Docs

This folder contains the Mintlify-ready documentation project for Shadow.

## Structure

- `docs.json`: Mintlify site configuration and navigation
- `index.mdx`: docs home page
- `quickstart.mdx`: fastest path to first success
- `getting-started/`: installation and onboarding docs
- `journeys/`: opinionated end-to-end user journeys
- `features/`: core product features and workflow pages
- `integrations/`: integration-specific guides
- `features/`: product tour and screenshots
- `settings/`: operational guidance
- `support/`: FAQ and troubleshooting
- `images/`: docs assets and screenshots used by the pages

## Recommended Mintlify setup

If this repository is connected to Mintlify as a monorepo, point the documentation path to:

`/mintlify-docs`

That path contains the `docs.json` file Mintlify needs to build the docs site.
